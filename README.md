Byju's-Sales-Analysis-Dashboard-2021-2023-
Overview This is a simple Power BI dashboard created to analyze Byju’s sales performance from 2021 to 2023. It helps understand sales trends, regional performance, product sales, and sales representative performance through clear visualizations.  No DAX formulas were used. The dashboard is built using basic charts and design features.
Tools Used
- Power BI
- Microsoft Excel
